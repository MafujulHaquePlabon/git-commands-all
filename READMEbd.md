- ### প্রজেক্ট নামানো ও তৈরি করা

| কমান্ড | বর্ণনা |
| ------- | ----------- |
| `git init` | একটি লোকাল গিট রেপো শুরু করুন |
| `git clone ssh://git@github.com/[username]/[repository-name].git` | একটি রিমোট রেপোর লোকাল অনুলিপি তৈরি করুন |
